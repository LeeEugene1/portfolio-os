# 20. Vercel 배포 플랜

## 목표

웹앱을 Vercel에 배포합니다.

## 작업

- Vercel project root를 `apps/web`으로 설정.
- build command 확인.
- output 설정 확인.
- preview 배포.
- production 배포 또는 promote.

## 완료 기준

- production URL이 포트폴리오 웹앱을 로드합니다.
- URL이 PC와 모바일 브라우저에서 동작합니다.
- Flutter WebView가 로드할 준비가 됩니다.

## 검증

PC와 모바일에서 production URL을 엽니다.
