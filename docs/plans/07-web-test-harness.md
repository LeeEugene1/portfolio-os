# 07. 웹 테스트 하네스 플랜

## 목표

웹 테스트 하네스를 설치하고 설정합니다.

## 작업

- Vitest 추가.
- React Testing Library 추가.
- jsdom 또는 happy-dom 추가.
- Playwright 추가.
- 테스트 setup 파일 추가.
- 기본 config 파일 추가.

## 완료 기준

- `npm run test`로 Vitest를 실행할 수 있습니다.
- `npm run test:e2e`로 Playwright를 실행할 수 있습니다.
- React component 테스트를 위한 setup이 준비됩니다.

## 검증

```bash
cd apps/web
npm run test
npm run test:e2e
```
