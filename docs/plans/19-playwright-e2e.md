# 19. Playwright E2E 플랜

## 목표

핵심 사용자 흐름의 E2E 커버리지를 확장합니다.

## 작업

- PC viewport 테스트 추가.
- 모바일 viewport 테스트 추가.
- 기본 Portfolio 창 테스트.
- Store/cart 흐름 테스트.
- Calculator 흐름 테스트.
- Resume/Contact 흐름 테스트.

## 완료 기준

- 핵심 흐름이 Playwright로 검증됩니다.
- `npm run verify:full`이 로컬에서 통과합니다.

## 검증

```bash
cd apps/web
npm run verify:full
```
