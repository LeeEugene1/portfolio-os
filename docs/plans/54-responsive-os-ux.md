# 54. Responsive OS UX 개선 플랜

## 목표

웹/모바일 공통으로 OS 창 조작과 앱 탐색 경험을 개선합니다. 창 닫기만 있던 상태에서 최소화, 확대/복원, 하단 독 복원을 제공하고, 실행 중인 앱과 정렬 가능한 앱 아이콘을 명확하게 표시합니다.

## 작업

- 창 상태 모델에 최소화와 확대 상태를 추가합니다.
- 창 헤더에 닫기, 최소화, 확대/복원 컨트롤을 추가합니다.
- 최소화된 창은 화면에서 숨기고 하단 독 아이콘으로 복원할 수 있게 합니다.
- 하단 독은 앱별 축약 아이콘과 실행 중 상태 점을 표시합니다.
- 확대된 창에서도 하단 독은 접근 가능하게 유지합니다.
- 데스크톱 앱 아이콘 이동은 `@dnd-kit` draggable로 구현하고 drag overlay를 ghost 피드백으로 사용합니다.
- PC에서는 아이콘을 viewport 안에서 자유 좌표로 이동할 수 있게 합니다.
- 모바일/WebView 근사 viewport에서 창, 독, 아이콘이 겹치지 않도록 스타일을 조정합니다.

## 완료 기준

- 사용자는 창을 닫기, 최소화, 확대/복원할 수 있습니다.
- 최소화된 앱은 하단 독에서 다시 열 수 있습니다.
- 실행 중인 앱은 하단 독 상태 점으로 구분됩니다.
- 전체화면 또는 모바일 화면에서도 하단 독으로 다른 앱을 확인할 수 있습니다.
- 앱 아이콘 이동 중 ghost 피드백이 표시되고, 이동 좌표가 전체 viewport 안으로 제한됩니다.
- 주요 viewport에서 창 헤더와 하단 독이 겹치지 않습니다.
