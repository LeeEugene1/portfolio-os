# 06. 웹 스캐폴드 플랜

## 목표

`apps/web`에 Next.js 앱을 생성합니다.

## 작업

- TypeScript와 App Router 기반 Next.js 생성.
- Tailwind CSS 추가.
- 로컬 dev server 실행 확인.
- production build 확인.

## 완료 기준

- `apps/web/package.json`이 존재합니다.
- `apps/web/app` 또는 `apps/web/src/app`이 존재합니다.
- `npm run dev`로 앱을 실행할 수 있습니다.
- `npm run build`가 성공합니다.

## 검증

```bash
cd apps/web
npm run build
```
