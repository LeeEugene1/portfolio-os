# 56. OS UX 후속 개선 플랜

## 목표

#54 이후 남은 OS UX를 정리합니다. 모바일 상단 터미널 탭, 좌우 스크롤 아이콘, 다중 창 스택, 풀사이즈 빈 여백 제거, PC 아이콘 겹침 방지, 창 생성 오프셋과 스택 애니메이션을 구현합니다.

## 작업

- 공통 풀사이즈 창 하단 빈 여백을 제거합니다.
- 모바일 앱 아이콘 영역을 좌우 스크롤 가능한 한 줄 구조로 변경합니다.
- 모바일 풀페이지 창에서는 상단 터미널 탭을 좌우 스크롤로 표시합니다.
- 모바일 탭에서 현재 앱, 열린 앱, 닫힌 앱을 시각적으로 구분합니다.
- 모바일 다중 창 기본 상태에서는 내부 상하 스크롤을 잠그고, 클릭한 창만 풀사이즈로 전환해 내부 스크롤을 허용합니다.
- 모바일에서 여러 창이 켜켜이 쌓인 카드처럼 보이게 합니다.
- PC 아이콘을 이동할 때 다른 아이콘과 겹치면 가까운 빈 좌표로 보정합니다.
- PC 창 생성 오프셋을 적용해 새 창이 기준 좌표에서 10px씩 어긋나게 열리도록 합니다.
- `framer-motion`으로 창 등장, 포커스 전환, 모바일 스택 전환을 가볍게 처리합니다.

## 완료 기준

- PC와 모바일에서 풀사이즈 창 하단에 불필요한 빈 여백이 없습니다.
- 모바일 아이콘 목록은 좌우 스크롤로 접근 가능합니다.
- 모바일 풀페이지 상태에서 상단 터미널 탭이 표시되고 현재 탭이 구분됩니다.
- 열린 앱과 닫힌 앱은 탭의 상태 점 컬러로 구분됩니다.
- 모바일에서 여러 창이 열리면 스택처럼 보이고, 클릭한 창만 풀사이즈로 내부 스크롤이 가능합니다.
- PC에서 아이콘 드롭 후 아이콘끼리 겹치지 않습니다.
- PC에서 여러 창을 열면 창 생성 오프셋과 스택 애니메이션으로 자연스럽게 쌓입니다.
